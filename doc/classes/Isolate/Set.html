write_fasta(type, file = nil)

Writes a FASTA file with sequences based on the different types: gene (3’->5’ sequence), CDS (the coding sequences; i.e. the reverse complement if necesary), protein (tranlated from the CDS) or 6frame (6 possible translations based upon frame and strand).

_check_result(res, exitonfail = true, msg = "an error has occurred")

Checks the result of a tool and exits if it fails and the argument exitonfail is true (default). It uses term/ansicolor for fancy terminal coloring.

Arguments:

res:result from an execute method
exitonfail:[true, false] controls wheter the funcion should terminate the running process on fail.
msg:Error message to append to error report.

splicing()

This methods returns the splicing pattern in gene coordinates. In other words, it is used to extract the exonic sequence (and potentially the introns) from the gene sequence.

[Source]

     # File genome.rb, line 398
398:         def splicing
399:             loc =[]
400:             
401:             case strand
402:             when 1
403:                 exons = oitems
404:             when -1
405:                 exons = oitems.reverse
406:             end
407:             
408:             f0 = exons[0].from
409:             exons.each do |exon|
410:                 x0 = exon.from - f0 + 1
411:                 x1 = exon.to - f0 + 1

413:             end
414:             loc = loc.join(",")
415:             loc = "join(" + loc + ")" if length > 1

splicing_original()

This methods displays the original splicing pattern based on the genomic coordinates. It can be used to extract the exons from the genome sequence. It is intended mainly as a check method.

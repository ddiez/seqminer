splicing()

This methods returns the splicing pattern in gene coordinates. In other words, it is used to extract the exonic sequence (and potentially the introns) from the gene sequence.

[Source]

     # File genome.rb, line 390
390:         def splicing
391:             loc =[]
392:             
393:             case strand
394:             when 1
395:                 exons = oitems
396:             when -1
397:                 exons = oitems.reverse
398:             end
399:             
400:             f0 = exons[0].from
401:             exons.each do |exon|
402:                 x0 = exon.from - f0 + 1
403:                 x1 = exon.to - f0 + 1
404:                 loc << x0.to_s + ".." + x1.to_s
405:             end
406:             loc = loc.join(",")
407:             loc = "join(" + loc + ")" if length > 1
408:             loc = "complement(" + loc + ")" if strand == -1
409:             loc
410:         end

splicing_original()

This methods displays the original splicing pattern based on the genomic coordinates. It can be used to extract the exons from the genome sequence. It is intended mainly as a check method.
